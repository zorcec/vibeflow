Add `@vibeflow-tools/ui-prototyping` React package for in-app variant switching. Provides `useVariant`, `VariantProvider`, `PageVariantSwitcher`, `VariantSwitcher`, and `VariantDevToolbar` components with URL param and localStorage persistence, keyboard shortcuts (Alt+H / Ctrl+Shift+V), and zero runtime dependencies.
