Page-level variant — Kanban Board layout

KanbanBoard

The dark segmented bar (top-left) switches between full-page layouts. Try Columns, Swimlane, and Compact. The active variant is synced to the URL — copy and share the link!

Component-level variant — TaskCard view density

TaskCard

Hover over each card to reveal the numbered dot switcher on the right. Switch between default (1), compact (2), and detailed (3). Each card uses the same variant state — they're all in the TaskCard scope.

Usage — React code example
// 1. Wrap your app with VariantProvider import { VariantProvider, useVariant, PageVariantSwitcher, VariantSwitcher, VariantDevToolbar } from '@vibeflow-tools/ui-prototyping' function App() { return ( <VariantProvider> <VariantDevToolbar /> // ⚡ floating toolbar (Ctrl+Shift+V) <KanbanBoard /> </VariantProvider> ) } // 2. Page-level switching (dark segmented bar, top-left) const layoutVariants = { columns: { direction: 'row' }, swimlane: { direction: 'column' }, compact: { direction: 'row', dense: true }, } function KanbanBoard() { const layout = useVariant('KanbanBoard', layoutVariants) return ( <> <PageVariantSwitcher name="KanbanBoard" variants={layoutVariants} /> <div className={`board board-${layout.direction}`}> <TaskCard /> </div> </> ) } // 3. Component-level switching (numbered dots, hover to reveal) const cardVariants = { default: {}, compact: { compact: true }, detailed: { showMeta: true, showComments: true }, } function TaskCard({ task }) { const variant = useVariant('TaskCard', cardVariants) return ( <div style={{ position: 'relative' }}> <VariantSwitcher name="TaskCard" variants={cardVariants} /> {variant.compact ? <CompactView /> : <FullView />} </div> ) }
URL Persistence

Shareable variant links

Variant state is synced to URL query params automatically:

// Active URL after switching KanbanBoard → swimlane, TaskCard → compact: ?vf[KanbanBoard]=swimlane&vf[TaskCard]=compact

✅ Shareable — send the URL, recipient sees the same variant
✅ Bookmarkable — preserved in browser history
✅ localStorage fallback — persists across sessions
